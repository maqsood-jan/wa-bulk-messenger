# WhatsApp Bulk Messenger

A simple web app to send bulk WhatsApp messages with Excel/CSV upload.

## Features
- QR code login (no phone number or API key needed)
- Upload Excel or CSV file
- 3 message templates (Balance, Recovery, Sale)
- Human-like sending with random delays
- Live progress and logs
- Export report to Excel

## Deploy to Render.com (Free)

1. Push this folder to your GitHub repository
2. Go to https://render.com and sign up free
3. Click "New Web Service"
4. Connect your GitHub repo
5. Set build command: `npm install`
6. Set start command: `npm start`
7. Click Deploy

That's it! You get a free URL like https://wa-bulk-messenger.onrender.com

## Usage
1. Open your app URL in any browser
2. Scan QR code with WhatsApp
3. Upload your Excel file
4. Write or select message template
5. Click Send
